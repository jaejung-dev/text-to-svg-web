What this page shows

The production Text-to-SVG service receives each validation prompt and returns raw SVG. The result is displayed between the original ground truth and the existing Claude, Gemini, and GPT-5.2 outputs so that design quality, prompt adherence, and visual structure can be reviewed without reading metrics first.
