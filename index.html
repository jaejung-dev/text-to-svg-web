Method

What this page shows

The production Text-to-SVG service receives each validation prompt and returns raw SVG using each model's deployed defaults. The results are displayed beside the original ground truth and existing Claude, Gemini, and GPT-5.2 outputs, then ranked by LicaScore for quick validation. Reference images also show the scorer values from the LicaScore report, with Qwen-8B fine-tuned output labeled as LicaScore.
